# `foundations/` — explainer pages

Plain-English explainers for the four projects this cookbook is built around. Each page answers three questions in the same order:

1. **What is it?**
2. **Why does it exist?**
3. **How is it different from the neighbor you already know?**

Plus one **"What nobody tells you"** callout per page — the contrarian or non-obvious observation.

## Pages (Wave 1)

- [what-is-openclaw.md](what-is-openclaw.md) — the 210K-star breakout of 2026
- [what-is-openshell.md](what-is-openshell.md) — NVIDIA's sandbox runtime
- [what-is-nemoclaw.md](what-is-nemoclaw.md) — NVIDIA's OpenClaw-on-Nemotron stack
- [what-is-hermes-agent.md](what-is-hermes-agent.md) — NousResearch's self-improving agent

## Scope (deliberately narrow)

This is not an encyclopedia of every agent project. These four are the specific substrate this cookbook builds on — if you understand them, the recipes and `core/` library make sense. For a broader landscape view, see [`../comparisons/`](../comparisons/).
