# `comparisons/` — framework and protocol face-offs

These pages exist because "X vs Y" is one of the highest-intent queries developers make. When you've decided you need a multi-agent framework, you don't want an essay — you want a table.

## Pages (Wave 2)

- `crewai-vs-langgraph-vs-autogen.md` — "CrewAI tutorial" alone is ~14,800 searches/month; this page competes for the comparison slice of that traffic
- `openclaw-vs-nemoclaw.md`
- `hermes-vs-openclaw.md`
- `openshell-vs-e2b-vs-docker-sandbox.md`
- `mcp-vs-tool-calling.md`
- `visual-builders-compared.md` — Langflow vs Dify vs Flowise vs n8n
- `rag-sota-2026.md` — published alongside `core/rag/` v1

## Format

Every comparison page has:
- A clear, opinionated recommendation up top ("If you need X, use Y")
- A comparison matrix with 5–10 dimensions
- One "What nobody tells you" callout
- Links to the specific recipe(s) in [`../recipes/`](../recipes/) that demonstrate each item
