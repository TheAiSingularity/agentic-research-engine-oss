# `core/tools/` — tool registry + MCP helpers

Building blocks for tool-calling agents:
- A lightweight registry that every framework variant in a recipe can share
- MCP (Model Context Protocol) server and client helpers — Python and Rust
- Tool schema patterns (OpenAI / Anthropic tool-use formats, JSON Schema, MCP)
- Tool allow-listing helpers (maps to HermesClaw's sandbox policies)
