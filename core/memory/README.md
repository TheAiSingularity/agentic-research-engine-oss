# `core/memory/` — persistent memory

Patterns for agents that need to remember across sessions. Inspired by Mem0's approach — but framework-agnostic, so any recipe can drop it in.

## Scope

- Short-term working memory (conversation state)
- Long-term episodic memory (what the user asked about yesterday)
- Long-term semantic memory (facts the user has shared)
- Memory consolidation (summarizing old memories instead of letting them pile up)
